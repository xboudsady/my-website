This document outlines the steps for setting up and configuring Maximo Mobile within the Maximo Application Suite (MAS) 9 environment.

## Prerequisites

* MAS 9 deployed and running.
* Maximo Manage and Mobile Applications deployed.

## Key Configuration Steps

1.  **Verify User Access:** Ensure users have the required security groups and application access within Maximo Manage.
2.  **Mobile Properties:** Review and configure system properties related to mobile sync and logging.
3.  **App Customization:** Utilize the Mobile Application Builder to customize the look and feel of the application.
4.  **Deployment:** Deploy the custom package to the MAS environment.

## Troubleshooting Common Issues

If users experience sync errors, check the application server logs and ensure the Mobile cron tasks are running.
